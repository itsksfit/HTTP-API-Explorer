# HTTP API Explorer

A command-line tool to test and inspect HTTP APIs.

## Features

- ✅ Send HTTP GET requests
- ✅ Display HTTP status codes
- 🚧 Parse JSON responses
- 🚧 Display response headers
- 🚧 Measure response time
- 🚧 Support POST, PUT and DELETE requests

## Tech Stack

- Python 3.11
- Requests
